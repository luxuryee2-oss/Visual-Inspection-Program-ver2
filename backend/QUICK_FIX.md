# 🔴 긴급: 서버 실행 안내

## 현재 문제

브라우저 주소창에 `localhost:3001`이 보입니다.
하지만 **서버는 포트 3000에서 실행**됩니다!

## 해결 방법

### 1단계: 서버 실행

**터미널**에서 다음 명령어 실행:

```bash
cd "C:\cursorstudy\Visual Inspection Program ver2\backend"
npx ts-node src/server-simple.ts
```

### 2단계: 서버 시작 확인

터미널에 다음과 같은 메시지가 표시되어야 합니다:

```
🚀 간단한 서버 시작...
서버가 포트 3000에서 실행 중입니다.
테스트: http://localhost:3000/api/inspection/direct-test
```

**이 메시지가 보이면** → 서버가 실행된 것입니다! ✅

### 3단계: 올바른 URL로 접속

**중요:** 포트는 **3000**입니다 (3001이 아님!)

브라우저 주소창에 다음을 입력:

```
http://localhost:3000/api/inspection/direct-test
```

## ⚠️ 확인 사항

1. **서버가 실행 중인가요?**
   - 터미널에 "서버가 포트 3000에서 실행 중입니다." 메시지가 보이나요?

2. **올바른 포트를 사용했나요?**
   - ✅ `http://localhost:3000/...`
   - ❌ `http://localhost:3001/...` (잘못됨!)

3. **터미널에 오류가 있나요?**
   - 오류 메시지가 있으면 전체 내용을 복사해서 알려주세요

## 빠른 확인

1. 터미널에서 서버 실행
2. "서버가 포트 3000에서 실행 중입니다." 메시지 확인
3. 브라우저에서 `http://localhost:3000/api/inspection/direct-test` 접속

## 여전히 안 되는 경우

터미널에서 서버를 실행했을 때:
- 어떤 메시지가 표시되나요?
- 오류 메시지가 있나요?
- 서버가 시작되었다는 메시지가 보이나요?

이 정보를 알려주세요!
