# 🚨 지금 바로 서버 실행하기

## 현재 상황

`ERR_CONNECTION_REFUSED` 오류 = **서버가 실행되지 않았습니다**

## ⚡ 빠른 해결 방법

### 단계 1: 터미널에서 서버 실행

**새로운 터미널 창**을 열고 (또는 기존 터미널 사용):

```bash
cd "C:\cursorstudy\Visual Inspection Program ver2\backend"
npx ts-node src/server-simple.ts
```

### 단계 2: 서버 시작 메시지 확인

터미널에 다음 메시지가 **반드시** 표시되어야 합니다:

```
🚀 간단한 서버 시작...
서버가 포트 3000에서 실행 중입니다.
테스트: http://localhost:3000/api/inspection/direct-test
```

**이 메시지가 보이면** → 서버가 정상적으로 실행된 것입니다! ✅

### 단계 3: 브라우저에서 테스트

서버가 실행된 상태에서 브라우저에서 접속:

```
http://localhost:3000/api/inspection/direct-test
```

**예상 응답:**
```json
{"message":"작동합니다!","timestamp":"2024-..."}
```

## ⚠️ 중요 확인 사항

### 서버가 시작되지 않는 경우

터미널에 **오류 메시지**가 표시됩니다. 예를 들어:

- `Cannot find module...` → 패키지 설치 필요
- `Port already in use` → 포트가 이미 사용 중
- 기타 오류 → 오류 메시지 전체 복사

**오류 메시지를 복사해서 알려주세요!**

### 서버는 시작되었는데 브라우저에서 안 되는 경우

1. **포트 확인**: 
   - 서버는 포트 **3000**에서 실행됩니다
   - 브라우저에서 `http://localhost:3000/...` 로 접속해야 합니다

2. **서버 터미널 확인**:
   - 브라우저에서 접속 시 서버 터미널에 로그가 출력되나요?
   - 로그가 안 보이면 → 요청이 서버에 도달하지 못한 것
